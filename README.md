# imdb-parser
This is a simple script to read movies' titles and years from CSV, get their rating from IMDB and create TXT with a list, sorted by rating

## Dependencies

OMDB https://github.com/jvanbaarsen/omdb
CSV


## Usage

1. Update the CSV file name
2. Run through ruby
